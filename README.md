# Dashboard-widget-weather
Dashboard widget with current weather

## Requirements

+ Installed Dashboard app. 

## How to install 

1. Copy or clone or create sub module in Dashboard/client/app/widgets directory 
2. Build dashboard client app 

```bash
# from Dashboard/ 
$ npm run build 
```
